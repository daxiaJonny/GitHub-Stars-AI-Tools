React

React 是一个用于构建用户界面的 JavaScript 库。

特性

  • 声明式： React 让你轻松创建交互式 UI。为应用的每一个状态设计简洁的视图，当数据变化时 React 能高效更新并渲染合适的组件。
  • 组件化： 构建管理自身状态的封装组件，然后组合它们以构成复杂的 UI。
  • 一次学习，随处编写： 我们不假设你使用的技术栈，因此你可以在 React 中开发新特性而无需重写现有代码。

安装

React 从一开始就被设计为逐步采用，你可以根据需要使用尽可能少或尽可能多的 React：

npm install react react-dom

文档

你可以在 官方网站 找到 React 文档。查看 快速入门 页面了解概况。
